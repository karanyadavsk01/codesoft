# 🤖 FAQ Chatbot

A Rule-Based FAQ Chatbot built using Python and Streamlit that answers frequently asked questions using TF-IDF and Cosine Similarity.

## Features

- Rule-Based Responses
- FAQ Search
- Related Questions
- Confidence Score
- Download Chat History
- Clear Chat
- User-Friendly Interface

## Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-learn

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```
