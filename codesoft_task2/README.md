# 🎮 Tic Tac Toe AI

A Python-based Tic Tac Toe game with an unbeatable AI using the Minimax Algorithm.

## Features

- Human vs AI
- Minimax Algorithm
- Winner Detection
- Draw Detection
- Restart Game
- Scoreboard
- Tkinter GUI

## Technologies Used

- Python
- Tkinter

## Run

```bash
python app.py
```
