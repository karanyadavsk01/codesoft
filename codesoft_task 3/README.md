# 🎬 Movie Recommendation System

A Movie Recommendation System built using Python and Streamlit that recommends movies based on Genre, Language, and Rating.

## Features

- Genre Filter
- Language Filter
- Rating Filter
- Movie Search
- Top Rated Movies
- Random Movie Recommendation
- Interactive Streamlit UI

## Technologies Used

- Python
- Streamlit
- Pandas

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```
